# Pages

## Páginas Implementadas

### Autenticação
- **LoginPage** – Autenticação JWT com backend

### Dashboard
- **DashboardPage** – KPIs e métricas do sindicato

### Gestão
- **AssociadosPage** – Lista paginada de associados
- **AssociadoFormPage** – Cadastro e edição de associados
- **UsuariosPage** – Gestão de usuários do sistema
- **UsuarioFormPage** – Cadastro e edição de usuários

### Eleições
- **EleicoesPage** – Lista de eleições/enquetes
- **EleicaoFormPage** – Criação e edição de eleições
- **ResultadosEleicaoPage** – Visualização de resultados

### Relatórios
- **RelatoriosPage** – Central de relatórios
- **RelatorioVisualizarPage** – Visualização de relatórios
- **RelatoriosVotacaoPage** – Relatórios específicos de votação
- **RelatorioCartorialPage** – Relatórios cartoriais

### Configurações
- **ConfiguracaoSindicatoPage** – Dados e configurações do sindicato

### Perfil
- **PerfilPage** – Gerenciamento do perfil do usuário (edição de dados, alteração de senha, histórico de ações)
